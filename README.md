# Customer Intelligence Platform: Churn Prediction, Segmentation & Lifetime Value Forecasting

## Project Overview

This project demonstrates an end-to-end customer analytics solution built using SQL, Python, and Power BI.

The objective was to simulate customer transaction data, build a relational database, engineer customer-level features, and develop an interactive business intelligence dashboard focused on customer retention, customer lifetime value (LTV), and customer segmentation.

The project follows a modern analytics workflow combining data engineering, business intelligence, and predictive analytics principles.

## Business Problem

Customer retention is one of the most important drivers of long-term business growth.

This project aims to answer key business questions:

Which customers generate the highest lifetime value?
Which customers are at risk of churning?
How do purchasing behaviours differ across customer segments?
Which acquisition channels drive the most valuable customers?
How can businesses prioritise retention efforts based on customer value?
